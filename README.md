# Floral Nest

**Floral Nest** është një faqe web për tregtinë online të luleve dhe buqetave. Projekti ofron dizajn modern, vizualizim të bukur të produkteve, dhe seksione të organizuara si Home, About, Products, Reviews dhe Contact.

## Features

- **Home Page**: Prezantim i markës dhe mesazhit “Beauty in every petal”.  
- **About Us**: Seksion me video dhe tekst që tregon filozofinë dhe arsyet pse të zgjedhësh Floral Nest.  
- **Products**: Shfaqje e produkteve me foto, çmim, zbritje, dhe ikonat për të shtuar në cart, pëlqyer dhe share.  
- **Reviews**: Seksion i klientëve me ratings dhe komentet e tyre.  
- **Contact**: Formë kontakti për mesazhe, me dizajn të thjeshtë dhe border pink.  
- **Responsive Design**: Faqja përshtatet për desktop, tablet dhe mobile.  
- **Visual Effects**: Hover, zoom në imazhe, hije të buta dhe borders të veçanta.
